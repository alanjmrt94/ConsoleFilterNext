 PROJECT LINEAGE

-----------------------------------------------------
 VERSION 4.0.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - In-game editor: paginated filter lists, regex validation, import/export, embedded presets
 - Add modIdFilters (filter logs by Forge mod id)
 - Add skipMessagesWithStackTrace option
 - Per-filter-type hit statistics (/consolefilter status)
 - /consolefilter profile now persists activeProfile to TOML
 - ConfigEditorModel round-trip tests; CI dedicated server smoke job
 - Migration guide from original ConsoleFilter (MIGRATION.md)

-----------------------------------------------------
 VERSION 3.5.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - Add full in-game config editor (Forge Mod List Config button, no extra dependencies)
 - Edit booleans, active/editing profiles, and all filter lists from the game UI
 - Save & Apply writes TOML and hot-reloads filters at runtime

-----------------------------------------------------
 VERSION 3.4.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - Add loggerFilters (legacy alias, merged with source filtering)
 - Add filter profiles: default, debug, production
 - Add /consolefilter export, import, and profile commands
 - Add filterLatestLog option for latest.log file appenders
 - Register client config screen (Forge Mod List / config button)
 - Pin ForgeGradle to 6.0.47; enable Gradle daemon for local dev

-----------------------------------------------------
 VERSION 3.3.1 - 1.20.1
-----------------------------------------------------
1.20.1:
 - threadFilters use contains matching (aligned with sourceFilters)
 - Refactor Log4jFilter to extend AbstractFilter

-----------------------------------------------------
 VERSION 3.3.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - Add /consolefilter reload, list, and status commands (OP level 2)
 - Add ignoreCase and whitelistMode config options
 - Track and report filtered message count
 - Switch to Parchment mappings (Librarian plugin) for improved IDE support

-----------------------------------------------------
 VERSION 3.2.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - sourceFilters use contains matching; levelFilters are case-insensitive
 - Precompile regex patterns on config load
 - Migrate to FMLJavaModLoadingContext constructor (Forge 47.4+ API)
 - Add Minecraft/Forge dependency ranges to mods.toml
 - Remove redundant Log4j/SLF4J dependencies from build
 - Add unit tests for FilterEntry and LogMessage
 - Rename dev-env.sh to release.sh; auto-generate run/eula.txt for runServer

-----------------------------------------------------
 VERSION 3.1.0 - 1.20.1
-----------------------------------------------------
1.20.1:
 - Wire JavaFilter, Log4jFilter, SystemOutFilter, and SystemErrFilter
 - Remove manual config file creation; use ForgeConfig only
 - Support config hot-reload via ModConfigEvent
 - Filter plain-text console lines (basic/regex) without log format parsing
